Tissot US Website Clone
Project Overview

This project is a front-end clone of the Tissot US official website created for educational and practice purposes.
The main goal of the project was to recreate the design, layout, and user interface of the original website while improving front-end development skills.

The project focuses on replicating the visual structure of the website including navigation menus, product sections, banners, and responsive layout.

Note: This project is not affiliated with or endorsed by Tissot. It is only a learning project.# WebTechnologies_1
